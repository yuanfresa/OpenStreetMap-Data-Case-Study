Districts

Stockholm Municipality is divided into 14 boroughs. The boroughs are subdivided into districts.

Top 10 districts with most nodes information
 
xxxxxxxxxx
SELECT tags.VALUE as CITY, COUNT(*) as count 
FROM  (SELECT * FROM NODE_TAGS ) tags 
WHERE tags.KEY='city'
GROUP BY tags.VALUE
ORDER BY COUNT(*) DESC
LIMIT 10;

Data Exploration

After having an overview of the data. I would like to investigate something related with recycling since Scandinavia countries are famous for their environmentally friendly recycling system.

  1. How many kinds of recycling?

     
    xxxxxxxxxx
    SELECT count(*)
    FROM (SELECT KEY as category
    FROM NODE_TAGS
    WHERE TYPE='recycling' AND VALUE='yes'
    GROUP BY KEY) as recycle

    Amazing, there are 33 categories in total for recycling

  2. What categories are for recycling? 

     
    xxxxxxxxxx
    SELECT KEY as category, count(*)
    FROM NODE_TAGS
    WHERE TYPE='recycling' AND VALUE='yes'
    GROUP BY KEY
    ORDER BY count(*) DESC;

  3. How many nodes have recycling station?

     
    xxxxxxxxxx
    SELECT count(ID)
    FROM NODE_TAGS WHERE KEY='amenity' and VALUE='recycling';

    716

  4. How many ways have recycling nodes?

     
    xxxxxxxxxx
    SELECT count(WAY_NODES.ID)
    FROM WAY_NODES
    JOIN (SELECT ID FROM NODE_TAGS WHERE KEY='amenity' and VALUE='recycling') as NODE_ID
    ON WAY_NODES.NODEID= NODE_ID.ID

    6

    Additional Ideas

    Data missing

    After we explore how many nodes and ways have recycling services, it is surprising that we can only find 6 ways having recycling service, which is not in accordance with 716 recycling nodes, which means not all nodes are recorded in the way tags, thus resulting in incomplete "way_nodes.csv". It will be bad to analysis the relationship between nodes and ways. For example, the node does not have information like max_speed in particular period which the way can provide, if we want to go to one restaurant by car, then it will be better to know this information from the way that this node locates.
